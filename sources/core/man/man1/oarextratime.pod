=head1 NAME

oarextratime - request some extra time for job handled by the OAR batch scheduler.

=head1 SYNOPSIS

B<oarextratime> <job_id>

=head1 DESCRIPTION

Add a request to get some extra time for a job (increase its walltime).

=head1 OPTIONS

=over

=item B<-V, --version>

Print OAR version number.

=item B<-h, --help>

Print help command message.

=back

=head1 SEE ALSO

oarsub(1), oardel(1) oarstat(1)
