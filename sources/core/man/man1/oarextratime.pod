=head1 NAME

oarextratime - request some extra time for a job handled by the OAR batch scheduler.

=head1 SYNOPSIS

B<oarextratime> <job_id>

=head1 DESCRIPTION

This command allows users to request some extra time for a job (increase of the walltime).

Once the request is registered, it will by handled during the next pass of scheduler, and granted if it fits with other jobs.

=head1 OPTIONS

=over

=item B<-V, --version>

Print OAR version number.

=item B<-h, --help>

Print help command message.

=back

=head1 WARNING

oarextratime is not compatible with suspending jobs at runtime (oarhold --running).
