Configuration
First configure the variables in variable.js file present in the Web Portal Directory. Configure the varibles to point to the location of apis.
If you are using port redirection use http://localhost:portnumber in variable.js.
For detailed configuration read User Manual and FAQ's section present in the help page.


Authentication
The Portal will ask for UserName and Password as soon as it loads. Kindly enter valid username and password as only authenticated users can perform certain actions.
Some users have already been created by default in oar_devel appliance. In Case, You are using oar_devel appliance Kindly use those username and passwords.
Find how to create new users and detailed information about Authentication go through User Manual.
