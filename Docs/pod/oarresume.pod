=head1 NAME

oarresume - resume job in OAR batch scheduler.

=head1 SYNOPSIS

B<oarhold> idjob

=head1 DESCRIPTION

This command is used to resume a job in OAR batch scheduler. Job is designed by its job's identifier.

oarsub(1), oardel(1) oarstat(1), oarnodes(1), oarhold(1)
