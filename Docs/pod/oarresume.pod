=head1 NAME

oarresume - resume a job in OAR batch scheduler.

=head1 SYNOPSIS

B<oarresume> job_id

=head1 DESCRIPTION

This command is used to resume a job in OAR batch scheduler. Job is designed by its job's identifier.

=head1 OPTIONS

=over

=item B<-v, --version>

Print OAR version number.

=item B<-h, --help>

Print help command message.
