=head1 NAME

oarnodes - show occupation and state of compute nodes.

=head1 SYNOPSIS

B<oarnodes> [ B<-a> | B<-s> | B<-l> | B<-h> ] [list of nodes]

=head1 DESCRIPTION

This command is used to the state and occupation of compute nodes in OAR batch scheduler.

=head1 OPTION

=over

=item B<-a, --all>

Show all nodes with their properties

=item B<-s, --state>

Show only node states

=item B<-l, --list>

Show only node list

=item B<-h, --help>

Show help usage
