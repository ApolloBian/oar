=head1 NAME

oarstat - show job's informations.

=head1 SYNOPSIS

B<oarstat> [B<-a>] [B<-s>] [B<-f>] [B<-x>] [B<-j> idJob] [B<-h> "date_start,date_stop" [B<-d>]]

=head1 DESCRIPTION

This command is used to show information of job in OAR batch scheduler.

=head1 OPTIONS

=over

=item B<-a>
print more informations

=item B<-s>
give some stats about state of nodes

=item B<-f>
detailed display

=item B<-x>
like -f

=item B<-j> idJob;
print detailed informations from a specific job

=item B<-h> "date_start,date_stop"
print history of jobs and state of nodes between two dates.
(you can use B<-d> option to print a dumper format)

=back
