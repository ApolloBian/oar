=head1 NAME

oarhold - hold a job in OAR batch scheduler.

=head1 SYNOPSIS

B<oarhold> job_id

=head1 DESCRIPTION

Ask OAR to not schedule job_id until oarresume command will be executed.

=head1 OPTIONS

=over

=item B<-vr, --running>

Manage not only Waiting jobs but also Running one (can suspend the job)

=item B<-v, --version>

rint OAR version number.

=item B<-h, --help>

Print help command message.

oarsub(1), oardel(1) oarstat(1), oarnodes(1), oarresume(1)
