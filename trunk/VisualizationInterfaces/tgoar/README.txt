                            tgoar
                            =====


What is it ?
============

tgoar is a basic web interface for OAR batch scheduler.
Originaly developed for PIPOL (http://gforge.inrialpes.fr/projects/pipol) project.


This is a TurboGears (http://www.turbogears.org) project. It can be
started by running the start-tgoar.py script.
