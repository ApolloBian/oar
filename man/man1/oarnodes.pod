=head1 NAME

oarnodes - show information about nodes managed by OAR.

=head1 SYNOPSIS

B<oarnodes> [ -a | -s | -r <integer> | --sql <sql_properties> | -l | -h | -V ] [<list of nodes>] [ -e [<date>] | -X | -Y | -D ]

=head1 DESCRIPTION

Display information about nodes managed by OAR.

=head1 OPTION

=over

=item B<-a, --all>

Show all resources with their properties.

=item B<-r, --resource>

Show only a resource with their properties.

=item B<-s, --state>

Show only resource states.

=item B<-l, --list>
     
Show only node list.

=item B<-e, --events>

Print events related to a node: either the last 20 if no date is given or all events since the specified date.

=item B<--sql>

Display resources which matches this sql where clause (ex: "state = 'Suspected'")

=item B<-D, --DUMPER>

Print result in DUMPER format.

=item B<-X, --XML>

Print result in XML format.

=item B<-Y, --YAML>

Print result in YAML format.

=item B<-V, --version>

Print OAR version number.

=item B<-h, --help>

Show help usage.

=back
